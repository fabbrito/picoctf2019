## Solution:

The cipher is a rotation which depends on the key.

```python
cipher = "UFJKXQZQUNB"
key = "SOLVECRYPTO"
```

Using the first letter as an example: with a key of "S", we take the rotation related to this letter, which is 18, then reverse it in the cipher "U"
This in simple python code is:

```python
print(ord("S")-ord("A")) # 18
print(ord("U")-18) # 67
print(chr(67)) # C
```
